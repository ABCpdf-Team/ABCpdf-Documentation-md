# JSContext Class

JSContext encapsulates the JavaScript environment.

``` System.Object WebSupergoo.ABCpdf13.JavaScript.JSContext ```

## Method Description &nbsp;ConvertToJS Gets a JavaScript value for the parameter value. &nbsp;CreateObject Creates a JavaScript object. &nbsp;DisposeOfCallback Disposes of a delegate used to create a JavaScript function. &nbsp;GetFieldPrototype Gets the Field prototype object. &nbsp;

## Property Description Doc Gets the PDF document on which JavaScript code runs. Options Gets the options for running JavaScript code. JSUndefined Gets the JavaScript undefined value. JSNull Gets the JavaScript null value. JSTrue Gets the JavaScript true value. JSFalse Gets the JavaScript false value. GlobalObject Gets the JavaScript global object.
