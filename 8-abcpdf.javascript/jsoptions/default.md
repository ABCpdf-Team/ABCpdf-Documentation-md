# JSOptions Class

JSOptions specifies options for running JavaScript code.

``` System.Object WebSupergoo.ABCpdf13.JavaScript.JSOptions ```

## Method Description none &nbsp; &nbsp;

## Property Description LoadGlobals Gets or sets whether to create certain predefined global variables and execute some JavaScript code defined in the document.

## Event Description SetUp Occurs after the JavaScript engine is initialized and before the specified JavaScript code is executed.
