# GetNumberInt Method

Gets the Int32 value of this number.

## Syntax

[C#]

```csharp
int GetNumberInt()
```

[Visual Basic]

```vb
Function GetNumberInt() As Integer
```

## Params

| **Name** | **Description** |
| --- | --- |
| return | The Int32 value. |

## Notes

The method returns the Int32 value of this number. It fails if this value is not a number.
