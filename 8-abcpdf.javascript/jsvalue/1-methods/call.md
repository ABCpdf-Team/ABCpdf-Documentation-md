# Call Method

Calls this function.

## Syntax

**[C#]**

```csharp
JSValue Call(IList args)
```

<span class=language>[Visual
            Basic]</span>  
`Function Call(args As IList(Of JSValue)) As JSValue``may throw Exception()`

## Params

| Name | Description | 
| --- | --- |
| args | A list of arguments. The first item is the 'this' argument. | 
| return | The return value of the function call. | 

## Notes

The method calls this value as a function.
