# GetString Method

Gets the String value of this string.

## Syntax

[C#]

```csharp
string GetString()
```

[Visual Basic]

```vb
Function GetString() As String
```

## Params

| **Name** | **Description** |
| --- | --- |
| return | The String value. |

## Notes

The method returns the String value of this string. It fails if this value is not a string.
