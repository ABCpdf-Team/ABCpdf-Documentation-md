# GetBooleanBool Method

Gets the Boolean value of this Boolean value.

## Syntax

[C#]

```csharp
bool GetBooleanBool()
```

## Params

| **Name** | **Description** |
| --- | --- |
| return | The Boolean value. |

## Notes

The method returns the Boolean value of this Boolean value. It fails if this value is not a Boolean value.
