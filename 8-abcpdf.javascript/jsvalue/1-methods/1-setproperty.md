# SetProperty Method

Sets this object's property.

## Syntax

[C#]

```csharp
void SetProperty(string name, JSValue v)
```

## Params

| **Name** | **Description** |
| --- | --- |
| name | The name of the property. |
| v | A new value of the property. |

## Notes

The method sets the property with the specified name to the new value.
