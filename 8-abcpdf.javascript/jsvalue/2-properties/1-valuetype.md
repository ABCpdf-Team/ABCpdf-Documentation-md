# ValueType Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp JSValue.Type ``` [Visual Basic] `JSValue.Type` | n/a | Yes | The type of the value. | 

## Notes

The JSValue.Type enumeration can take any of the following values:

- Undefined
- Null
- Number
- String
- Boolean
- Object
- Function
- Error
- Array
- Symbol
- ArrayBuffer — ArrayBuffer object
- TypedArray — typed array object
- DataView — DataView object
