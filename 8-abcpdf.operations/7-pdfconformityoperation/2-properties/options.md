# Options Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp PdfConformityOperationOptions ``` [Visual Basic]`PdfConformityOperationOptions` | None | No | The PDF update options. | 

## Notes

Options related to the way that the PDF may be changed as part of the operation[](conformance.md).

It can take a combination of the following values:

- None – no option.
- KeepDocumentModDate – do not update the document modification date if it exists.
