# Messages Property

| **Type** | **Default** | **Read Only** | **Description** |
| --- | --- | --- | --- |
| [C#] <BR> `IList&lt;string&gt;` | null | Yes | The changes made to make the document conformant. |

## Notes

To make a document conformant, it may need to be changed.

After the document has been changed, this property contains a list of the changes made to the document.
