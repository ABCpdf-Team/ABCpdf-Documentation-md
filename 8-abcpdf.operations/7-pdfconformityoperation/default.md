# PdfConformityOperation Class

Operation to produce conforming PDF files.

Note that this feature is only available under the ABCpdf Professional License.

``` System.Object WebSupergoo.ABCpdf13.Operations.PdfConformityOperation Implements: IDisposable ```

## Method Description &nbsp;GetData Get the conforming document as raw data. &nbsp;GetStream Get the conforming document as raw data stream. &nbsp;Save Write the conforming PDF document. &nbsp;

## Property Description Conformance The PDF conformance to be applied. Conformity The PDF conformity options. ExcludedConformity The PDF conformity options to exclude. Options The PDF update options. Messages The changes made to make the document conformant. Errors The features within the document which could not be made conformant.
