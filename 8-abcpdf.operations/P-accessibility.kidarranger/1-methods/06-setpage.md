# SetPage Function

Sets the Page for this structure element.

## Syntax

[C#]

```csharp
SetPage(this StructureElementElement element, Page page)
```

[Visual Basic]

```vb
SetPage(this StructureElementElement element, Page page)
```

## Params

| **Name** | **Description** |
| --- | --- |
| page | The new page. |

## Notes

Sets the Page for this structure element.
