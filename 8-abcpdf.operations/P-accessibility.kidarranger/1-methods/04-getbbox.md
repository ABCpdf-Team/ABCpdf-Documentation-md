# GetBBox Function

Get the bounding box for the structure element.

## Syntax

[C#]

```csharp
GetBBox(this StructureElementElement element)
```

## Params

| **Name** | **Description** |
| --- | --- |
| return | The bounding box. |

## Notes

Get the bounding box for the structure element.

Will return null if there is no bounding box.
