# ShowClippedText Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | true | No | Whether to show text which is invisible because it is affected by a clip path. | 

## Notes

Whether to show text which is invisible because it is affected by a clip path.

By default all text is returned. This includes text that may not be visible to the user because it has been affected by a clipping path. Using this setting you control whether you want this text returned by [GetText](../1-methods/gettext.md).
