# TextOperation Property

| **Type** | **Default** | **Read Only** | **Description** |
| --- | --- | --- | --- |
|  | n/a | Yes | The text analysis operation for the accessibility analysis. |

## Notes

The text analysis operation for the accessibility analysis.

You can adjust the way that the AccessibilityOperation collects text by adjusting the TextOperation properties before triggering analysis.
