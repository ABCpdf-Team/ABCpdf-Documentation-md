# ImageProperties Class

A class that represents one raster image (bitmap) in a document.

``` System.Object WebSupergoo.ABCpdf13.Operations.ImageProperties ```

## Method Description none &nbsp; &nbsp;

## Property Description PixMap The PixMap object associated with the image. Renditions The set of renditions of the image at different locations within the document. Dpi The resolution of the image in dots per inch. DpiX The horizontal resolution of the image in dots per inch. DpiY The vertical resolution of the image in dots per inch.
