# Seen Function

Determines if the object provided has previously been processed by this operation.

## Syntax

[C#]

```csharp
bool Seen(IndirectObject io)
```

## Params

| **Name** | **Description** |
| --- | --- |
| io | The object to test against. |
| return | Whether the object has been seen. |

## Notes

Determines if the object provided has previously been processed by this operation.
