# Alpha Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 1.0 | No | Sets the backdrop alpha | 

## Notes

When objects are compositied with the backdrop, this alpha value will be the minimum alpha of objects after compositing
