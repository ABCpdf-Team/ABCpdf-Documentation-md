# Dpi Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 0.0 | No | In the case of vector content one needs to render the page before presenting the image for OCR. | 

## Notes

In the case of vector content one needs to render the page before presenting the image for OCR.

This setting determines the DPI at which the page should be rendered.

A high value allows you to detect smaller text but takes longer to process.
