# WordColor Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp XColor ``` [Visual Basic] `XColor` | null | No | Whether to make detected words visible using a particular color. | 

## Notes

Whether to make detected words visible using a particular color. The default is null which results in invisible, but selectable, text.

None.
