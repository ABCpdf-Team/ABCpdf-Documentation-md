# ConfidenceColor Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp XColor ``` [Visual Basic] `XColor` | null | No | The confidence color is a vertical bar indicating the reliability of the OCR for each piece of text. | 

## Notes

The confidence color is a vertical bar indicating the reliability of the OCR for each piece of text.

This property allows you to select the color to use. The default is null which means no confidence bar will be drawn.
