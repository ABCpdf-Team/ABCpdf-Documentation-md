# Abort Function

Abort the current processing.

## Syntax

[C#]

```csharp
virtual void Abort()
```

## Params

| **Name** | **Description** |
| --- | --- |
| none |  |

## Notes

Abort the current processing.

All remaining operators will be skipped.

You can use this function to prematurely terminate the processing of a content stream.
