# Y Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double? ``` [Visual Basic]`Nullable(Of Double)` | n/a | Yes | Gets the y-coordinate of the location of the source object. | 

## Notes

This property holds the vertical offset of the location.

Values are in the coordinate system of the source object.
