# UseSourceRect Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | false | No | Whether to draw only the portion of the page specified by the Souce | 

## Notes

Whether to draw only the portion of the page specified by the Souce.Rect. If this value is false then the entire visible page will be drawn. Default false.
