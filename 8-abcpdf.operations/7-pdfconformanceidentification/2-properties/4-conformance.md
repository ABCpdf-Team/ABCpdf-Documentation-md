# Conformance Property

| **Type** | **Default** | **Read Only** | **Description** |
| --- | --- | --- | --- |
| [C#] <BR> `string` | null | No | The conformance level. |

## Notes

This property indicates the conformance level. For PDF/A-1a, the value is "A"; for PDF/A-1b, the value is "B".
