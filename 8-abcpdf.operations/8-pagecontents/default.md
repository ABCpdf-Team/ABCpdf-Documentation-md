# PageContents Class

Operation to analyse and manipulate the text in a set of pages.

``` System.Object WebSupergoo.ABCpdf13.Operations.PageContents ```

## Method Description &nbsp;PageContents PageContents Constructor. AddLayers Add a particular set of Layer objects from a particular Page. &nbsp;AddPages Add pages to be processed. &nbsp;

## Property Description &nbsp;IncludeColor Whether to include color information in the output. &nbsp;IncludeAnnotations Whether to include annotation and field text in the output. &nbsp;RegenerateUnicode Whether to attempt to regenerate missing Unicode tables from embedded fonts.
