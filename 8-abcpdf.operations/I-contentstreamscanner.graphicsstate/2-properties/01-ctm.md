# CTM Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp XTransform ``` [Visual Basic] `XTransform` | null | Yes | The Current Transformation Matrix (CTM). | 

## Notes

The Current Transformation Matrix (CTM).

This is used to map user coordinates - the values in the content stream - to device coordintes - positions on the page.
