# StrokeAdjustment Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | false | No | Whether to apply stroke adjustment or not. | 

## Notes

Whether to apply stroke adjustment or not.

Adjusted strokes maintain line thickness at the expense of errors in positioning.

None.
