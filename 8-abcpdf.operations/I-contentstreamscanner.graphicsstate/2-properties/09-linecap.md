# LineCap Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp LineCap ``` [Visual Basic] `LineCap` | null | No | The line cap for the end of stroked paths. | 

## Notes

The line cap for the end of stroked paths.

The LineCap enumeration may take the following values:

- Butt - Butt line cap.
- Round - Round line cap.
- Square - Square line cap.
