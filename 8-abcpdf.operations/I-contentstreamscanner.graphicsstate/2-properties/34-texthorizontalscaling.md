# TextHorizontalScaling Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 0.0 | No | A factor to be used to horizontally scale text. | 

## Notes

A factor to be used to horizontally scale text. A value of 1.0 indicates normal scaling.
