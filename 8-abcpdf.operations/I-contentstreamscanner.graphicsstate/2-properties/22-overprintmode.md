# OverprintMode Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | 0 | No | The overprint mode. | 

## Notes

The overprint mode. This may be 0 or 1 and the value controls the way that colors are interpreted when overprinting.
