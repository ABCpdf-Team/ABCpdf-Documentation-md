# DashArray Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double[] ``` [Visual Basic] `Double[]` | null | No | The dash array defining the lengths of shown and unshown parts of a path. | 

## Notes

The dash array defining the lengths of shown and unshown parts of a path.

The first length is shows, the second hidden and so on. When the end of the array is reached it repeats.

None.
