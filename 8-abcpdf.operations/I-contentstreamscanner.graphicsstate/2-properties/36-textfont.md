# TextFont Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | The name of the current font. | 

## Notes

The name of the current font.

This may be one of the pre-defined built in fonts like Times-Roman or Helvetica.

If it is not one of the pre-defined fonts then it indicates a font resource in the Fonts subdictionary.

## Example

None.
