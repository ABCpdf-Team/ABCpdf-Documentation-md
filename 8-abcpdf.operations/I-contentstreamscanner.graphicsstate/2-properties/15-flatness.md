# Flatness Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 0.0 | No | The degree of precision to use when drawing curves. | 

## Notes

The degree of precision to use when drawing curves.

The value represents the maximum error which is acceptable and is measured in device units - typically pixels.

A value of zero represents a device default.

None.
