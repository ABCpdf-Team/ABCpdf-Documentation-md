# Font Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp FontObject ``` [Visual Basic] `FontObject` | n/a | Yes | The font object used for drawing this text fragment. | 

## Notes

The font object used for drawing this text fragment.

The font object can be used to find information about the font.
