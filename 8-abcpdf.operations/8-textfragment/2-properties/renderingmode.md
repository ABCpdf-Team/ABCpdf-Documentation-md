FillAndStroke - Fill text and stroke glyph outlines. Invisible - Neither fill nor stroke. FillAndClip - Fill text then add to clipping path. StrokeAndClip - Stroke text then add to clipping path. FillStrokeAndClip - Fill and stroke text then add to clipping path. Clip - Add to clipping path. |  |  |
