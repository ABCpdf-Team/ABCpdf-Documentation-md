# TextState Class

The text state which determines the positioning of text.

``` System.Object WebSupergoo.ABCpdf13.Operations.ContentStreamScanner.TextState ```

## Method Description TextState Create a new TextState initialized to default values. GetTextRenderingMatrix Get the current Text Rendering Matrix (Trm). Advance Advances the text position by an amount - typically the width of a glyph. &nbsp;

## Property Description TextMatrix The transformation matrix which defines the scale and positioning of a block of text. TextLineMatrix The transformation matrix which defines the scale and positioning of the current line of text within the current block of text.
