## Doc Obj

Next we create an ABCpdf Doc object and give it our basic settings.

We enlarge the line width, increase the font size, enable justification and inset the drawing rectangle from the edges of the document.

[C#] using var doc = new Doc(); doc.Width = 4; doc.FontSize = 32; doc.TextStyle.Justification = 1; doc.Rect.Inset(20, 20); [Visual Basic] Using doc As New Doc() doc.Width = 4 doc.FontSize = 32 doc.TextStyle.Justification = 1 doc.Rect.Inset(20, 20)

## Adding

We add our base text to the current page of the document. We then enter a loop, adding pages and chaining HTML boxes together until we run out of text to display.

[C#] doc.FrameRect(); id = doc.AddTextStyled(text); while (doc.Chainable(id)) { doc.Page = doc.AddPage(); doc.FrameRect(); id = doc.AddTextStyled("", id); } [Visual Basic] doc.FrameRect() theID = doc.AddTextStyled(theText) While doc.Chainable(theID) doc.Page = doc.AddPage() doc.FrameRect() theID = doc.AddTextStyled("", theID) End While

## Save

After adding all our text we save the document to a specified location and clear our document.

[C#] doc.Save(Server.MapPath("textflow.pdf")); [Visual Basic] doc.Save(Server.MapPath("textflow.pdf")) End Using
