# Equals Function

Test whether the two NullAtoms are the same

## Syntax

[C#]

```csharp
bool Equals(<a href="../default.htm">NullAtom</a> other)
```

[Visual Basic]

```vb
Function Equals(other As <a href="../default.htm">NullAtom</a>) As Boolean
```

## Params

| **Name** | **Description** |
| --- | --- |
| other | The NullAtom to test against. |

## Notes

Test whether the two NullAtoms are the same.

NullAtoms are are always equal to each other.
