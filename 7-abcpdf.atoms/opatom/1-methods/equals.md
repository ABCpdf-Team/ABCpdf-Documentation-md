# Equals Function

Test whether the two OpAtoms are the same

## Syntax

**[C#]**

```csharp
bool Equals(OpAtom other)
```

<span class=language>[Visual
            Basic]</span>  

            `Function Equals(other As OpAtom) As Boolean`
## Params

| Name | Description | 
| --- | --- |
| other | The OpAtom to test against. | 

## Notes

Test whether the two OpAtoms are the same.

Two OpAtoms are judged to be equal if their [Text](../2-properties/text.md) is equal.
