# BoolAtom Constructor

Construct a BoolAtom.

## Syntax

**[C#]**

```csharp
BoolAtom()
BoolAtom(bool value)
```

**[Visual Basic]**

```
Sub New()
Sub New(value As Boolean)
```

## Params

| Name | Description | 
| --- | --- |
| value | The initial value that the Atom should adopt. | 

## Notes

Create a BoolAtom.

If a value is not specified the default of false will be used.
