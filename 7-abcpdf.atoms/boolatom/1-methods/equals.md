# Equals Function

Test whether the two BoolAtoms are the same

## Syntax

[C#]

```csharp
bool Equals(<a href="../default.htm">BoolAtom</a> other)
```

## Params

| **Name** | **Description** |
| --- | --- |
| other | The BoolAtom to test against. |

## Notes

Test whether the two BoolAtoms are the same.

Two BoolAtoms are judged to be equal if their [Truth](2-properties/truth.md) is equal.
