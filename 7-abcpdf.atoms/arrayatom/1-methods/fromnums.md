# FromNums Function

Create an ArrayAtom of NumAtoms from a list of numbers.

## Syntax

[C#]

```csharp
static <a href="../default.htm">ArrayAtom</a> FromNums(IList&lt;double&gt; items)
```

[Visual Basic]

```vb
Shared Function FromNums(items as IList(Of Double)) As <a href="../default.htm">ArrayAtom</a>
```

## Params

| **Name** | **Description** |
| --- | --- |
| items | The list of numbers. |
| return | The resulting ArrayAtom. |

## Notes

Create an ArrayAtom of NumAtoms from a list of numbers.

If the provided list is null then the return value will be null.
