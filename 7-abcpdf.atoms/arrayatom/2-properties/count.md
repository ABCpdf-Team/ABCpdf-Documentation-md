# Count Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | n/a | Yes | The number of Atoms in the array. | 

## Notes

The number of Atoms in the array.

As Atoms are added to the array the Count will increase.
