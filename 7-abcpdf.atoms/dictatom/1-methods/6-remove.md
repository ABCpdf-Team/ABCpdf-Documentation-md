# Remove Function

Remove an element from the dictionary.

## Syntax

[C#]

```csharp
void Remove(string name)
```

[Visual Basic]

```vb
Sub Remove(name As String)
```

## Params

| **Name** | **Description** |
| --- | --- |
| name | The name of the element to be removed. |

## Notes

Remove the element with the specified name from the dictionary.
