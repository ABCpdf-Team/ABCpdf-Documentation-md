# Epsilon Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 0.00001 | Yes | The smallest value that can be written out to a PDF document. | 

## Notes

The smallest value that can be written out to a PDF document.

This value is based around the number of digits that can be written in the fractional part of a floating point number.
