# ValidAsInt64 Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | false | Yes | Whether the number is representable as a 64 bit integer. | 

## Notes

Whether the number is representable as a 64 bit integer. Real numbers may not be representable if they have floating point values or if they are too large or small.
