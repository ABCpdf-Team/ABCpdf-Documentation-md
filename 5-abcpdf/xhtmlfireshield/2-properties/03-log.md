# Log Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | "" | No | File access log for the worker process. | 

## Notes

File access log for the worker process.

In the event that the [Policy](01-policy.md) is set to Log, files to which access should be denied, are reported via this property.

None.
