# XSaveTemplateData Class

Represents some format-specific data not obtained from a template file.

``` System.Object WebSupergoo.ABCpdf13.XSaveTemplateData ```

## Method Description SetMeasureResolution Sets the measurement resolutions. &nbsp;

## Property Description ImageDisplaySmoothing The smoothing for displaying images. JpegQuality The JPEG quality level. MeasureDpiX The horizontal measurement resolution. MeasureDpiY The vertical measurement resolution. ReencodeJpeg Whether JPEG images are re-encoded in JPEG. &nbsp; &nbsp;
