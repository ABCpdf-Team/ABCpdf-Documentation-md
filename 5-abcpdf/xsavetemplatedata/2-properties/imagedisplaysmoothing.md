| This property specifies the smoothing for displaying images. It can take any of the following values: Default – specifies the default algorithm. Alias – specifies no anti-aliasing. AntiAlias – specifies anti-aliasing. Some output formats support different smoothing types for displaying images. For SWF, this affects the FillStyleType field of the FILLSTYLE structure for images. |  |  |
