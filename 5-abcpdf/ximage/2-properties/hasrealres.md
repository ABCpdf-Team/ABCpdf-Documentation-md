# HasRealRes Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic]`Boolean` | See description. | Yes | Whether the image specifies the resolution. | 

## Notes

This property indicates whether [HRes](hres.md) and [VRes](vres.md) are obtained from the image. [HRes](hres.md) and [VRes](vres.md) contain default values if they are not obtained from the image.
