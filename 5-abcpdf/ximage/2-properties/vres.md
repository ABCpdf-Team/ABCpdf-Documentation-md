# VRes Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | See description. | Yes | The vertical resolution of the current frame (DPI). | 

## Notes

This property reflects the vertical resolution of the current frame. It is measured in dots per inch (DPI).
