# Type Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `String` | See description. | Yes | The type of image. | 

## Notes

The type property reflects the type of image that has been loaded using [SetFile](../1-methods/setfile.md) or [SetData](../1-methods/setdata.md). It can take one of the following values.

- ""
- "BMP"
- "EMF"
- "EXIF"
- "GIF"
- "Icon"
- "JPEG"
- "MemoryBMP"
- "PNG"
- "TIFF"
- "WMF"
- "EPS"
- "PS"
- "SWF"
