# UseEmbeddedHalftone Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | True | No | Whether to use embedded halftones. | 

## Notes

Specifies whether to use embedded halftones.

If embedded halftones are not used then the [DefaultHalftone](defaulthalftone.md) will be used for all halftoning.

## Example

See the [DefaultHalftone](defaulthalftone.md) property.
