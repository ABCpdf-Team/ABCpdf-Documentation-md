# PreserveK Property

| Type | Default | Read Write | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `bool` | false | Yes | Determines if color transforms preserve black when rendering to CMYK. | 

## Notes

Tell the color management system whether or not to preserve black (K) when rendering to CMYK.

If true the [PreserveKOptions](preservekoptions.md) property method is followed. If false it is not.
