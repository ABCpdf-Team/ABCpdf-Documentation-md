# PreMultiply Function

Pre-multiplies this transformation matrix by the supplied transform.

## Syntax

[C#]

```csharp
void PreMultiply(<a href="../default.htm">XTransform</a> transform)
```

[Visual Basic]

```vb
Sub PreMultiply(transform as <a href="../default.htm">XTransform</a>)
```

## Params

| **Name** | **Description** |
| --- | --- |
| transform | The transform to combine with this one. |

## Notes

Pre-multiplies this transformation matrix by the supplied transform.

The final result is contained in this transform.

See also the [Mutiply](postmultiply.md) function.

None
