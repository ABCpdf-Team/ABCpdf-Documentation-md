# StreamCryptionMethod Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp CryptMethodType ``` [Visual Basic] `CryptMethodType` | V2 | Yes | The crypt method for streams for encryption levels of type 4 or above. | 

## Notes

This property returns the crypt methods for streams. Crypt method settings are in effect only when [Type](type.md) is 4 or above.

To change this value, use [SetCryptMethods](../1-methods/setcryptmethods.md).
