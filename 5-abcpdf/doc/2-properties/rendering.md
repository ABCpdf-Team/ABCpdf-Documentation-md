# Rendering Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp XRendering ``` [Visual Basic] `XRendering` | n/a | No | The rendering options and control. | 

## Notes

This property allows you control over the way that PDFs are rendered.

Note that Rendering is only available under the ABCpdf Professional License.

See the [XRendering](../../xrendering/default.md) object for further details.
