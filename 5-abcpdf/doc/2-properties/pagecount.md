# PageCount Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | 0 | Yes | The number of pages in the document. | 

## Notes

This property holds the number of pages in the document.

You can add pages to the document using the [AddPage](../1-methods/addpage.md) method.
