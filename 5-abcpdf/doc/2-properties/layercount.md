# LayerCount Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | n/a | Yes | The number of layers on the current page. | 

## Notes

This property holds the number of layers on the current page.

You can add layers to pages using calls to methods such as [AddText](../1-methods/addtext.md) and [AddImage](../1-methods/addimage.md). See the [Layer](layer.md) property for more details.
