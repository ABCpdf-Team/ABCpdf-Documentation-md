# Clear Function

Clears the document.

## Syntax

[C#]

```csharp
void Clear()
```

[Visual Basic]

```vb
Sub Clear()
```

## Params

| **Name** | **Description** |
| --- | --- |
| none |  |

## Notes

Use this method to release resources and return the document to a just-created state.
