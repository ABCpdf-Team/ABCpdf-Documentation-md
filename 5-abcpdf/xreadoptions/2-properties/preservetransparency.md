# PreserveTransparency Property

| **Type** | **Default** | **Read Only** | **Description** |
| --- | --- | --- | --- |
|  | false | No | Gets or sets a value that indicates whether the transparency of raster images should be preserved. |

## Notes

This property is used when the source is a raster image. If the method (e.g. [Doc.AddImageObject](doc/1-methods/addimageobject.md)) called has a parameter for specifying the same setting, the parameter overrides this property.
