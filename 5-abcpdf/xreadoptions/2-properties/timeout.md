# Timeout Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic]`Integer` | 60000 | No | Gets or sets the timeout in milliseconds for external components. | 

## Notes

Timeout is used for external components, such as the XPS printer driver for [ReadModule](1-readmodule.md) of XpsAny.
