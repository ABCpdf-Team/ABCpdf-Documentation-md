# MSOfficeParameters 
Property

| **Type** | **Default** | **Read Only** | **Description** |
| --- | --- | --- | --- |
| [C#] <BR> `Dictionary&lt;string, string&gt;` | n/a | No | MS Office PDF conversion control parameters. |

## Notes

This property is used by the MS Office import module. It is used to pass custom parameters to Microsoft Office for precise control over the PDF conversion process.

The parameters are a set of named strings. You can find details of the options you can use on the MS Word, Excel and Publisher web pages.
