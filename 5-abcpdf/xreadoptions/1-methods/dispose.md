# Dispose Function

Dispose of the object.

## Syntax

[C#]

```csharp
void Dispose()
```

[Visual Basic]

```vb
Sub Dispose()
```

## Params

| **Name** | **Description** |
| --- | --- |
| none |  |

## Notes

You can call this method to explicitly dispose of an object and reduce the garbage collection overhead.

This method disposes of the [Operation](2-properties/operation.md).

Do not attempt to use an object after calling Dispose.
