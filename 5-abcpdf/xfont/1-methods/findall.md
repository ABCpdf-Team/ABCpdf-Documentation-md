# FindAll Function

Find all the fonts currently installed on the system.

## Syntax

[C#]

```csharp
static <a href="../default.htm">XFont</a>[] FindAll()
```

## Params

| **Name** | **Description** |
| --- | --- |
| return | The set of matching fonts. |

## Notes

This function finds all the fonts currently installed on the system.
