# XPalette Class

A palette of different colors that may be used.

This class allows you to specify how to select colors when image colors are reduced, for example when exporting as an indexed color image.

## Property Description Definition The type of palette. Data Raw bytes containing palette data. Dither Whether to dither colors when mapping to the palette. FixedColors Fixed colors that should be in the palette. Size The target number of colors in the palette.
