# FixedColors Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp IList ``` [Visual Basic] `IList` | An empty list. | Yes | Fixed colors that should be in the palette. | 

## Notes

Fixed colors that should be in the palette.

Add fixed colors to the palette when you want certain colors to be present in the palette.

This is only relevant when the [Definition](01-definition.md) is Adaptive or Exact.

None.
