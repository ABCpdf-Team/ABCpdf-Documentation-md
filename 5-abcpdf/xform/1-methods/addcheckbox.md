# AddCheckbox Function

Add check box form field.

## Syntax

[C#]

```csharp
Field AddCheckbox(XRect rect, string name, bool value)
```

## Params

| **Name** | **Description** |
| --- | --- |
| rect | Rectangle of the form field |
| name | Name of the form field |
| value | Initial value |
| return | Form field object |

## Notes

Add check box form field.
