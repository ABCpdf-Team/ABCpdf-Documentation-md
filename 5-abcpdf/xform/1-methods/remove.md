# Remove Function

Remove a field from the form.

## Syntax

[C#]

```csharp
void Remove(Field field)
```

[Visual Basic]

```vb
Sub Remove(field As Field)
```

## Params

| **Name** | **Description** |
| --- | --- |
| field | The field to be removed. |

## Notes

Remove a field from the form.
