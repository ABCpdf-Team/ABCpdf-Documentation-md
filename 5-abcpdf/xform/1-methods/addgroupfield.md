# AddGroupField Function

Add group field.

## Syntax

**[C#]**

```csharp
Field AddGroupField(List kids, string name, string value)
```

<span class=language>[Visual Basic]</span>  

            `Function AddGroupField(kids As List, name As string, value As string) As Field`
			
## Params

| Name | Description | 
| --- | --- |
| kids | The kids to go in the form field | 
| name | Name of the form field | 
| value | Initial value for kids | 
| return | Form field id | 

## Notes

Add group field.
