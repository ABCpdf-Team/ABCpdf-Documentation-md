# AddComboBoxField Function

Add combo box form field.

## Syntax

[C#]

```csharp
Field AddComboBoxField(XRect rect, string name)
```

[Visual Basic]

```vb
Function AddComboBoxField(rect As XRect, name As string) As Field
```

## Params

| **Name** | **Description** |
| --- | --- |
| rect | Rectangle of the form field |
| name | Name of the form field |
| return | Form field object |

## Notes
