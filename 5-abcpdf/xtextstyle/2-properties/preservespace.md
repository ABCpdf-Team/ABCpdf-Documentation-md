# PreserveSpace Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp bool ``` [Visual Basic] `Boolean` | false | No | Whether to preserve white space characters for plain text. | 

## Notes

This property determines whether white space characters are preserved when [Doc.AddText](../../doc/1-methods/addtext.md) is called. It does not affect [Doc.AddTextStyled](../../doc/1-methods/addtextstyled.md).
