# GetFonts Function

Gets all the fonts in this document.

## Syntax

**[C#]**

```csharp
FontObject[] GetFonts()
```

**[Visual Basic]**

`Function GetFonts() As FontObject()`
## Params

| Name | Description | 
| --- | --- |
| return | All the fonts in the document. | 

## Notes

This function scans the document and builds an array of all the fonts.

## Example

None.
