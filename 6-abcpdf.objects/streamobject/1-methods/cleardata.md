# ClearData Function

Clear the data and compression settings for the stream.

## Syntax

**[C#]**

```csharp
void ClearData()
```

**[Visual Basic]**

`Sub ClearData()`
## Params

| Name | Description | 
| --- | --- |
| none |  | 

## Notes

Clears all data and compression settings leaving the stream empty of data.
