# CertificateCollection Class

Represents a collection of X509Certificate2 objects.

``` System.Object WebSupergoo.ABCpdf13.Objects.Signature.CertificateCollection ```

## Method Description &nbsp;CertificateCollection CertificateCollection Constructor. &nbsp;Add Adds the certificate object to this collection if its value is not already present. &nbsp;AddOrImportRange Adds the certificate objects or imports the certificates from the files to this collection for those whose values are not already present. &nbsp;AddRange Adds the certificate objects to this collection for those whose values are not already present. Clear Removes all contained certificate objects. &nbsp;Contains Determines whether this collection contains the certificate object's value. GetHashCode Calculates a hash value based on the values of the contained certificate objects. &nbsp;Import Imports the certificates from the file for those whose values are not already present. &nbsp;ImportAt Imports at the specified position the certificates from the file for those whose values are not already present. &nbsp;IndexOf Gets the index of the certificate object's value. &nbsp;Insert Inserts the certificate object to this collection if its value is not already present. &nbsp;InsertRange Inserts the certificate objects to this collection for those whose values are not already present. &nbsp;RemoveAt Removes the certificate object at the specified index. &nbsp;

## Property Description Count The number of certificate objects.
