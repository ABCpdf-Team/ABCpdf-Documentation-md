# SquareAnnotation Function

Add square annotation to the current page of the doc.

## Syntax

**[C#]**

```csharp
SquareAnnotation(Doc doc, XRect rect, XColor borderColor, XColor fillColor)
```

<span class=language>[Visual Basic]</span>  

            `SquareAnnotation(doc As Doc, rect As XRect, borderColor As XColor, fillColor As XColor)`			
## Params

| Name | Description | 
| --- | --- |
| doc | Doc | 
| rect | Annotation rectangle | 
| borderColor | Border color | 
| fillColor | Fill color | 

## Notes

Add square annotation to the current page of the doc.

## Example

None.
