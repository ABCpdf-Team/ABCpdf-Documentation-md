# LineAnnotation Function

Add line annotation to the current page of the doc.

## Syntax

[C#]

```csharp
<a href="../default.htm">LineAnnotation</a>(Doc doc, XPoint p1, XPoint p2, XColor color)
```

[Visual Basic]

```vb
<a href="../default.htm">LineAnnotation</a>(doc As Doc, p1 As XPoint, p2 As XPoint, color As XColor)
```

## Params

| **Name** | **Description** |
| --- | --- |
| doc | Doc. |
| p1 | Line start point. |
| p2 | Line end point. |
| color | Color of the line. |

## Notes

Add line annotation to the current page of the doc.

None
