# ContentHeight Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | n/a | Yes | The content height of the image in pixels. | 

## Notes

The content height of the image. It is usually very close to [ScrollHeight](scrollheight.md).

When adding HTML you only see one page at a time. This property allows you to find the height of the HTML before it is paged onto the PDF.
