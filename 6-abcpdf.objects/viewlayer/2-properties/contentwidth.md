# ContentWidth Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | n/a | Yes | The content width of the image in pixels. | 

## Notes

The content width of the image. It is usually very close to [ScrollWidth](scrollwidth.md).

When adding HTML you only see one page at a time. This property allows you to find the width of the HTML before it is paged onto the PDF.
