# FullText Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `String` | n/a | Yes | The full text provided in the initial call to AddTextStyled or AddText. | 

## Notes

The full text provided in the initial call to AddTextStyled or AddText.

This text is shared between all the items in a chain. It is in plain text and does not indicate any styles.

None.
