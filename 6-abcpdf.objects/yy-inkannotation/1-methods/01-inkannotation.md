# InkAnnotation Function

Add ink annotation to the current page of the doc.

## Syntax

[C#]

```csharp
<a href="../default.htm">InkAnnotation</a>(Doc doc, XRect rect)
```

[Visual Basic]

```vb
<a href="../default.htm">InkAnnotation</a>(doc As Doc, rect As XRect)
```

## Params

| **Name** | **Description** |
| --- | --- |
| doc | Doc |
| rect | Annotation rectangle |

## Notes

Add ink annotation to the current page of the doc.
