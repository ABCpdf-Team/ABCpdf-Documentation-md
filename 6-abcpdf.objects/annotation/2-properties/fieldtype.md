# FieldType Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `String` | n/a | Yes | The field type for any form field associated with this annotation. | 

## Notes

The field type for any form field associated with this annotation.

The following field types are available:

- "Pushbutton"
- "Checkbox"
- "Radio"
- "Text"
- "List"
- "Combo"
- "Signature"
- "Unknown"

More details of these field types can be found in the Section 8.6.3 of the [Adobe PDF Specification](http://partners.adobe.com/).
