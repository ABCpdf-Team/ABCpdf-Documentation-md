# Color Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp XColor ``` [Visual Basic] `XColor` | n/a | Yes | The Color of the annotation. | 

## Notes

The Color of the annotation. This color is used for the following purposes.

- The background of the annotation’s icon when closed.
- The title bar of the annotation’s pop-up window.
- The border of a link annotation

## Example

None.
