# PopupAnnotation Function

Add popup annotation to the current page of the doc.

## Syntax

**[C#]**

```csharp
PopupAnnotation(Doc doc, XRect rect, int parentID)
```

<span class=language>[Visual Basic]</span>  

            `PopupAnnotation(doc As Doc, rect As XRect, parentID As int)`
			
## Params

| Name | Description | 
| --- | --- |
| doc | Doc | 
| rect | Annotation rectangle | 
| parentID | ID of the parent annotation | 

## Notes

Add popup annotation to the current page of the doc.

## Example

None.
