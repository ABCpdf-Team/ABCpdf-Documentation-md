# MacResFork Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp StreamObject ``` [Visual Basic] `StreamObject` | null | No | The Macintosh resource forks stream for this file | 

## Notes

The Macintosh resource forks stream for this file.

If there is no resource fork the value will be null.
