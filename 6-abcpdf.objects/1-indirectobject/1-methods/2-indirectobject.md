# IndirectObject Constructor

IndirectObject Constructor.

## Syntax

**[C#]**

```csharp
IndirectObject()
```

**[Visual Basic]**

`Sub New()`
## Params

| Name | Description | 
| --- | --- |
| none |  | 

## Notes

Creates an indirect object containing a [NullAtom](../../../7-abcpdf.atoms/nullatom/default.md).

Typically you will need to replace the NullAtom before you can do anything useful with the object.
