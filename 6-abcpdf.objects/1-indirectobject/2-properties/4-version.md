# Version Property

| Type | Default | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | 10 | No | The minimum version of the PDF specification required to support this object. | 

## Notes

The minimum version of the PDF specification required to support this object.

For example a version of 13 indicates that the features specified by the object require a PDF 1.3 capable browser. Adobe Acrobat 4 was the first browser to support PDF 1.3.
