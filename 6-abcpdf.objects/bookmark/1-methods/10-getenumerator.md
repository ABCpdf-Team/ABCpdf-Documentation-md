# GetEnumerator Function

Gets an enumerator for the Collection.

## Syntax

**[C#]**

```csharp
IEnumeratorBookmark> GetEnumerator()
```

**[Visual Basic]**

`Function GetEnumerator() As IEnumerator(Of Bookmark)`
## Params

| Name | Description | 
| --- | --- |
| return | The enumerator for the collection. | 

## Notes

Gets a Bookmark enumerator for the Collection.

## Example

None.
