# XRef Class

Cross-reference section.

``` System.Object WebSupergoo.ABCpdf13.Objects.ObjectSoup.XRef ```

## Method Description GetSubsectionsAsBytes Returns the cross-reference subsections as a byte array. GetSubsectionsAsString Returns the cross-reference subsections as a string. &nbsp;

## Property Description Type Gets the type of cross-reference section. Position Gets the byte offset of the cross-reference section. Trailer Gets the trailer dictionary. Stream Gets the cross-reference stream. Prev Gets the previous cross-reference section. XRefStm Gets the cross-reference stream for this cross-reference section in the hybrid-reference format.
