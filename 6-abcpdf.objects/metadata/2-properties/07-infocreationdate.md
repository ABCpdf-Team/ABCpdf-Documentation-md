# InfoCreationDate Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | The CreationDate entry within the Metadata | 

## Notes

The CreationDate entry within the [Metadata](../default.md).

This corresponds to the xmp:CreateDate entry.

Use [StringAtom.StringToDate](file:///C|/WSG/Projects/ABCpdf/Docs/PDF10000/Manual/7-abcpdf.atoms/stringatom/1-methods/stringtodate.md) and DateToString to convert between DateTime and string formats.
