# Equals Function

Test whether the two [Metadata](../default.md) objects are the same

## Syntax

**[C#]**

```csharp
virtual bool Equals(Metadata other)
```

<span class=language>[Visual Basic]</span>  

            `Overridable Function Equals(other As Metadata) As Boolean`
			
## Params

| Name | Description | 
| --- | --- |
| other | The Metadata to test against. | 
| return | Whether the Metadata objects are equal. | 

## Notes

Test whether the two [Metadata](../default.md) objects are the same.

None.
