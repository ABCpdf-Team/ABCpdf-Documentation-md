# Metadata Function

[Metadata](../default.md) constructor

## Syntax

**[C#]**

```csharp
Metadata(objectSoup soup)
Metadata(objectSoup soup, string path)
Metadata(objectSoup soup, byte[] data)
```

<span class=language>[Visual Basic]</span>  

```
Metadata(soup As ObjectSoup)
Metadata(soup As ObjectSoup, path As Stringpath)
Metadata(soup As ObjectSoup, data() As Byte)
```

## Params

| Name | Description | 
| --- | --- |
| soup | The ObjectSoup to contain the newly created Metadata. | 
| path | A path to a file containing data which should be placed in the Metadata. | 
| data | An array of bytes which should be placed in the Metadata. |
