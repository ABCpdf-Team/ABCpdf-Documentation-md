# CloudyEffect Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double ``` [Visual Basic] `Double` | 0.0 | No | Intensity of the cloudy border effect. | 

## Notes

Intensity of the cloudy border effect. Default value: 0.

If you change this value you will need to adjust the Annotation.Rect. See the [constructor](../1-methods/01-polygonannotation.md) for details.
