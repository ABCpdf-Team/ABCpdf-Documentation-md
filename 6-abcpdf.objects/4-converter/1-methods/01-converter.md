# Converter Function

Create an Atom converter for a particular ObjectSoup.

## Syntax

**[C#]**

```csharp
Converter(ObjectSoup soup)
```

<span class=language>[Visual Basic]</span>  

            `Converter(soup As ObjectSoup)`
			
## Params

| Name | Description | 
| --- | --- |
| soup | The soup to use. | 

## Notes

Create an Atom converter for a particular ObjectSoup.

None.
