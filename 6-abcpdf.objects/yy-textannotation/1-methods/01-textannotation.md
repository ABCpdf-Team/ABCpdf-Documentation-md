# TextAnnotation Function

Add text annotation to the current page of the doc.

## Syntax

[C#]

```csharp
<a href="../default.htm">TextAnnotation</a>(Doc doc, XRect rect1, XRect rect2, string contents)
```

[Visual Basic]

```vb
<a href="../default.htm">TextAnnotation</a>(doc As Doc, rect1 As XRect, rect2 As XRect, contents As string)
```

## Params

| **Name** | **Description** |
| --- | --- |
| doc | Doc |
| rect1 | Annotation rectangle |
| rect2 | Rectangle of the "sticky note" popup |
| contents | Contents of the text annotation |

## Notes

Add text annotation to the current page of the doc.
