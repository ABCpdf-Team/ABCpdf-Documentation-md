# Clear Function

Removes all items from the collection.

## Syntax

**[C#]**

```csharp
void Clear()
```

**[Visual Basic]**

`Sub Clear()``may throw NotSupportedException()`

## Params

| Name | Description | 
| --- | --- |
| none |  | 

## Notes

Removes all items from the collection.

All Fields collections are read only so calling this function will generate a NotSupportedException.
