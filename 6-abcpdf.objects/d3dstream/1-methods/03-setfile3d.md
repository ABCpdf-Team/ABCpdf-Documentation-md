# SetFile3D Function

Set the 3D content of the stream from data embedded in a file.

## Syntax

**[C#]**

```csharp
virtual void SetFile3D(string path)
```

<span class=language>[Visual Basic]</span>  

            `Overridable Function SetFile3D(path As string) As void`
			
## Params

| Name | Description | 
| --- | --- |
| path | A path to a file containing 3D data in OBJ, U3D or PRC format. | 

## Notes

Set the 3D content of the stream from data embedded in a file.

## Example

None.
