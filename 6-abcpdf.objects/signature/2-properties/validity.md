# Validity Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp Signature.ValidationInfo ``` [Visual Basic] `Signature.ValidationInfo` | null | Yes | The validity of the signature. | 

## Notes

The validation properties of the signature returned by the last call to [Validate](../1-methods/validate.md).

If Validate has not been called then this property will be null.
