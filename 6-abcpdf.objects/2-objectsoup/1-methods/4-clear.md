# Clear Function

Removes all objects from the Soup.

## Syntax

**[C#]**

```csharp
void Clear()
```

<span class=language>[Visual
            Basic]</span>  
`Sub Clear()`
## Params

| Name | Description | 
| --- | --- |
| none |  | 

## Notes

Removes all IndirectObjects from the Soup.

Only the Null object at array index zero is preserved.

None.
