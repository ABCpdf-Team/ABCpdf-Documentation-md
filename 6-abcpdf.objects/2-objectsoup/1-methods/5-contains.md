# Contains Function

Determines whether the Soup contains a specific object.

## Syntax

[C#]

```csharp
bool Contains(<a href="../../1-indirectobject/default.htm">IndirectObject</a> value)
```

[Visual Basic]

```vb
Function Contains(value As <a href="../../1-indirectobject/default.htm">IndirectObject</a>) As Boolean
```

## Params

| **Name** | **Description** |
| --- | --- |
| value | The object to locate. |
| return | True if the object is found, otherwise false. |

## Notes

Determines whether the Soup contains a specific IndirectObject.
