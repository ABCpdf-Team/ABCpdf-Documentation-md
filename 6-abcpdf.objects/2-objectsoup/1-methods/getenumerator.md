# GetEnumerator Function

Gets an enumerator for the Soup.

```csharp

```

**[Visual Basic]**

`Function GetEnumerator() As IEnumerator(Of IndirectObject)`
## Params

| Name | Description | 
| --- | --- |
| return | The enumerator for the collection. | 

## Notes

Gets an IndirectObject enumerator for the Soup.
