# Eof Class

Object representing file trailer and PDF end-of-file marker.

``` System.Object WebSupergoo.ABCpdf13.Objects.ObjectSoup.Eof ```

`Implements: IDisposable`

## Method Description Load Loads the file trailer data. &nbsp;

## Property Description IsLoaded Indicates whether this instance is loaded. Position Gets the byte offset of the end-of-file marker. StartXRef Gets the byte offset of the cross-reference section specified after the keyword startxref. XRef Gets the last cross-reference section.
