# Parent Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp Field ``` [Visual Basic] `Field` | See description. | Yes | The parent of this field. | 

## Notes

This property holds the parent of this field.

Each field may have a number of children which you can access using the [Kids](kids.md) property. Each child will have one parent which you can access using the Parent property. Fields at the top of the hierarchy will have a null parent.
