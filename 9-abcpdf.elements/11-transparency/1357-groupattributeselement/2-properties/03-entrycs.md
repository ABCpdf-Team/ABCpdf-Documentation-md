# EntryCS Property

## Notes

Represents the "CS" entry of the group attributes dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a ColorSpaceElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 147, page 350.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 145, page 419.
