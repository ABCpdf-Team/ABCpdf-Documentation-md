# EntryCS Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ColorSpaceElement ``` [Visual Basic] `ColorSpaceElement` | null | No | Represents the "CS" entry of the group attributes dictionary object. | 

## Notes

Represents the "CS" entry of the group attributes dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a [ColorSpaceElement](../../../08-graphics/0020-colorspaceelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 147, page 350.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=358)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 145, page 419.](https://www.iso.org/standard/63534.md)
