# Equals Function

## Syntax

[C#]

```csharp
virtual bool Equals(<a href="../default.htm">Element</a> other)
```

[Visual Basic]

```vb
Overridable Function Equals(other As <a href="../default.htm">Element</a>) As Boolean
```

## Params

| **Name** | **Description** |
| --- | --- |
| other | The Element to test against. |
| return | Whether the Elements are equal. |

## Notes

Test whether the two [Elements](default.md) are the same.
