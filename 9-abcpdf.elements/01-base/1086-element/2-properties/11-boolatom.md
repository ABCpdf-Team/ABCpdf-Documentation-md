# BoolAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp BoolAtom ``` [Visual Basic] `BoolAtom` | null | Yes | Gets the Atom property as a BoolAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as a BoolAtom.

If the [Atom](03-atom.md) is not a BoolAtom then the property will be null.

## Example

None.
