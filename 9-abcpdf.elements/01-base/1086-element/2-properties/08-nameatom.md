# NameAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp NameAtom ``` [Visual Basic] `NameAtom` | null | Yes | Gets the Atom property as a NameAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as a NameAtom.

If the [Atom](03-atom.md) is not a NameAtom then the property will be null.

## Example

None.
