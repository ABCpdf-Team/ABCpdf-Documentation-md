# EntryMetadata Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp MetadataStreamElement ``` [Visual Basic] `MetadataStreamElement` | null | No | This property represents the "Metadata" entry of the Element. | 

## Notes

This property represents the "Metadata" entry of the [Element](../default.md).

It is an optional entry for all Dictionaries or Streams as defined as part of the PDF 1.4 specification.

A [MetadataStreamElement](../../../14-document/1557-metadatastreamelement/default.md).

For definitive details see:.

[ISO 32000-1:2008 PDF 1.7; Table: 30, page 78.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=86)

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 3.27, page 23.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=23)
