# StringAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp StringAtom ``` [Visual Basic] `StringAtom` | null | Yes | Gets the Atom property as a StringAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as a StringAtom.

If the [Atom](03-atom.md) is not a StringAtom then the property will be null.

## Example

None.
