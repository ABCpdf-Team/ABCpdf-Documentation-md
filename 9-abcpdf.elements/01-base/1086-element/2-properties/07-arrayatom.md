# ArrayAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayAtom ``` [Visual Basic] `ArrayAtom` | null | Yes | Gets the Atom property as an ArrayAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as an ArrayAtom.

If the [Atom](03-atom.md) is not an ArrayAtom then the property will be null.

## Example

None.
