# NumAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp NumAtom ``` [Visual Basic] `NumAtom` | null | Yes | Gets the Atom property as a NumAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as a NumAtom.

If the [Atom](03-atom.md) is not a NumAtom then the property will be null.

## Example

None.
