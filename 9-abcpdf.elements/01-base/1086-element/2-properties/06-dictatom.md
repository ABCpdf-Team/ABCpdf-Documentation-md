# DictAtom Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp DictAtom ``` [Visual Basic] `DictAtom` | null | Yes | Gets the Atom property as a DictAtom. | 

## Notes

Gets the [Atom](03-atom.md) property as a DictAtom.

If the [Atom](03-atom.md) is not a DictAtom then the property will be null.

## Example

None.
