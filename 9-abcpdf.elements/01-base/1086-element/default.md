# Element Class

An Element is a specialization wrapper around an Atom designed to expose aspects of that Atom in a structured manner.

Different subclasses of Element are used to represent the different objects defined in the Adobe PDF Specification.

In general there is one Element subclass for each object table in the PDF Specification.

## Method Description &nbsp;Element Create a new Element. &nbsp;Assign Assign an IndirectObject to this Element. Equals Test whether the two Elements are the same. ClearCache Clears any cached Element items. &nbsp;

## Property Description Host Gets the host IndirectObject. Ref Gets any RefAtom that is part of this Element. Atom Gets any Atom that is part of this Element. RefOrAtom Gets the Atom that was used to create this Element. Object Gets the IndirectObject that is part of this Element. DictAtom Gets the Atom property as a DictAtom. ArrayAtom Gets the Atom property as an ArrayAtom. NameAtom Gets the Atom property as a NameAtom. NumAtom Gets the Atom property as a NumAtom. StringAtom Gets the Atom property as a StringAtom. BoolAtom Gets the Atom property as a BoolAtom. EntryMetadata This property represents the "Metadata" entry of the Element.
