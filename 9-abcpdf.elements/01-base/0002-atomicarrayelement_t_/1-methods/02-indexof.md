# IndexOf Function

Determines the first index matching a specific value.

## Syntax

[C#]

```csharp
virtual int IndexOf(T value)
```

## Params

| **Name** | **Description** |
| --- | --- |
| value | The value to locate in the Collection |
| return | If found, the index of value, otherwise -1. |

## Notes

Determines the first index matching a specific value.
