# ReportUnknownEntryName Function

Called to report an entry with an unrecognized name.

## Syntax

**[C#]**

```csharp
virtual void ReportUnknownEntryName(string entry)
```

<span class=language>[Visual Basic]</span>  

            `Overridable Function ReportUnknownEntryName(entry As string) As void`
			
## Params

| Name | Description | 
| --- | --- |
| entry | The entry name. | 

## Notes

Called to report an entry with an unrecognized name.
