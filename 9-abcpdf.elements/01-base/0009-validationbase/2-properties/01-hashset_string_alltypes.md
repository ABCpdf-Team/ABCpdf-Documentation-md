# HashSet&lt;string&gt; AllTypes Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp static ``` [Visual Basic] `static` | null | Yes | Gets all types that may be recorded in the Coverage. | 

## Notes

Gets all types that may be recorded in the [Coverage](02-coverage.md).

By removing covered types from the all types set, you can determine which types have not been covered.
