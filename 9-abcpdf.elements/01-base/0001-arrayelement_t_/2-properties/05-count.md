# Count Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | 0 | Yes | Gets the number of Elements contained in the array. | 

## Notes

Gets the number of [Elements](../../1086-element/default.md) contained in the array.

As [Elements](../../1086-element/default.md) are added to the array the Count will increase.

As they are removed, it will decrease.
