# IndexOf Function

## Syntax

[C#]

```csharp
virtual int IndexOf(T value)
```

## Params

| **Name** | **Description** |
| --- | --- |
| value | The Element to locate in the Collection |
| return | If found, the index of value, otherwise -1. |

## Notes

Determines the index of a specific [Element](1086-element/default.md).
