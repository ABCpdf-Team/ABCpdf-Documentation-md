# Count Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int ``` [Visual Basic] `Integer` | 0 | Yes | Gets the number of key/value pairs contained in the dictionary. | 

## Notes

Gets the number of key/value pairs contained in the dictionary.

As pairs are added to the array the Count will increase.

As they are removed, it will decrease.
