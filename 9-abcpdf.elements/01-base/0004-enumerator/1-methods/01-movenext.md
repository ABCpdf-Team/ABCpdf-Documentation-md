# MoveNext Function

Advances the enumerator to the next element of the [DictElement](../../0003-dictelement_t_/default.md).

## Syntax

**[C#]**

```csharp
virtual bool MoveNext()
```

<span class=language>[Visual Basic]</span>  

            `Overridable Function MoveNext() As Boolean`
			
## Params

| Name | Description | 
| --- | --- |
| return | true if the enumerator was successfully advanced to the next element; false if the enumerator has passed the end of the collection. | 

## Notes

Advances the enumerator to the next element of the DictElement.ValueCollection.

## Example

None.
