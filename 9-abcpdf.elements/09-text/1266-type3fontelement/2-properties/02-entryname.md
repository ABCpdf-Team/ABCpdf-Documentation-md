# EntryName Property

## Notes

Represents the "Name" entry of the type 3 font dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 112, page 258.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 110, page 314.
