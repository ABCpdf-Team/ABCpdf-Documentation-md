# EntryType Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | Represents the "Type" entry of the font dictionary object. | 

## Notes

Represents the "Type" entry of the font dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

This item must always be provided and must always take the value "Font".

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; page 254.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=262)
