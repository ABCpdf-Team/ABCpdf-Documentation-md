# EntryDescendantFonts Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementCIDFontElement> ``` [Visual Basic] `ArrayElementCIDFontElement>` | null | No | Represents the "DescendantFonts" entry of the type 0 font dictionary object. | 

## Notes

Represents the "DescendantFonts" entry of the type 0 font dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains [CIDFontElements](../../1277-cidfontelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 121, page 280.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=288)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 119, page 338.](https://www.iso.org/standard/63534.md)
