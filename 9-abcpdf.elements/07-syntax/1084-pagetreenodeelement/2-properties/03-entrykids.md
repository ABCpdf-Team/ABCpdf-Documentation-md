# EntryKids Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementElement> ``` [Visual Basic] `ArrayElementElement>` | null | No | Represents the "Kids" entry of the page tree node object. | 

## Notes

Represents the "Kids" entry of the page tree node object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains [PageTreeNodeElements](../default.md) or [PageObjectElements](../../1085-pageobjectelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 29, page 76.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=84)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 30, page 102.](https://www.iso.org/standard/63534.md)
