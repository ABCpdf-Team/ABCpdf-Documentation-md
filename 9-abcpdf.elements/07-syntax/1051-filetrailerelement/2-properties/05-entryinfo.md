# EntryInfo Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp DocumentInformationElement ``` [Visual Basic] `DocumentInformationElement` | null | No | Represents the "Info" entry of the file trailer dictionary object. | 

## Notes

Represents the "Info" entry of the file trailer dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [DocumentInformationElement](../../../14-document/1558-documentinformationelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 15, page 43.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=51)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 15, page 57.](https://www.iso.org/standard/63534.md)
