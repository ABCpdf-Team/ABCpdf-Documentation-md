# EntryD Property

## Notes

This property represents the "D" entry of the object.

It is a required entry and defined as part of the PDF 1.0 specification.

This property contains a DestinationElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; 12.3.2.3 Named Destinations, page 367.
