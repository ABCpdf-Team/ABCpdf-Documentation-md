| Represents the "PageLayout" entry of the catalog dictionary object. It is an optional entry defined as part of the PDF 1.0 specification. It contains a string representing a PDF name object. The PDF specification states that this item assumes a value of "SinglePage" if no value has been provided. This item may take one of the following valid values:. SinglePageOneColumnTwoColumnLeftTwoColumnRightTwoPageLeftTwoPageRight For definitive details see:. The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 28, page 73. Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 3.25, page 23. The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 29, page 99.
