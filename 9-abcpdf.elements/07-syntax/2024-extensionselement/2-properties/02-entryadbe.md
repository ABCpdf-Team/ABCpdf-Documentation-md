# EntryADBE Property

## Notes

Represents the "ADBE" entry of the extensions dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a DeveloperExtensionsElement.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 3.25a, page 24.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 48, page 140.
