# EntryRF Property

## Notes

Represents the "RF" entry of the full file specification dictionary object.

It is an optional entry defined as part of the PDF 1.3 specification.

It contains a RelatedFileSetElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 44, page 103.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 3.41, page 26.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 43, page 134.
