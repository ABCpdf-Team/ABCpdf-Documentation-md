# EntryType Property

## Notes

Represents the "Type" entry of the embedded file stream dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "EmbeddedFile".

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 45, page 104.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 44, page 136.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 244, page 460.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 247, page 558.
