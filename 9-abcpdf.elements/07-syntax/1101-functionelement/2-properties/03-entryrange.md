# EntryRange Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp IList ``` [Visual Basic] `IList` | null | No | Represents the "Range" entry of the function dictionary object. | 

## Notes

Represents the "Range" entry of the function dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains an array which contains doubles, representing PDF numeric objects.

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 38, page 93.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=101)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 38, page 123.](https://www.iso.org/standard/63534.md)
