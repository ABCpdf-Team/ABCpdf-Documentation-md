# EntryD Property

## Notes

Represents the "D" entry of the collection subitem dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

This property may contain one of two different types:.

1) A double representing a PDF numeric object.

2) A string representing a PDF string object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 49, page 107.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 47, page 139.
