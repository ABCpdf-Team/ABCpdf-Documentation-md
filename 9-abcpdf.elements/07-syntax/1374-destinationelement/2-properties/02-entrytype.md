# EntryType Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | This property represents the type of destination. | 

## Notes

This property represents the type of destination.

It is a required entry and defined as part of the PDF 1.0 specification.

It is a Name and may take one of the following valid values:.

- XYZ
- Fit
- FitH
- FitV
- FitR
- FitB
- FitBH
- FitBV

.For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 151, page 366.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=374)
