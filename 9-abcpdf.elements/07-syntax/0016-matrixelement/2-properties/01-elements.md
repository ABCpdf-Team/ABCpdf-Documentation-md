# Elements Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp IList ``` [Visual Basic] `IList` | null | No | The six elements in this list represent the following elements of the three by three transformation matrix:. | 

## Notes

The six elements in this list represent the following elements of the three by three transformation matrix:.

[a, b, 0].

[c, d, 0].

[e, f, 1].
