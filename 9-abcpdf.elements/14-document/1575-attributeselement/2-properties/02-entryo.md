| Represents the "O" entry of the attribute object dictionary object. It is a required entry defined as part of the PDF 1.0 specification. It contains a string representing a PDF name object. This item may take one of the following valid values:. LayoutListPrintFieldTableLink The specification defines this as exensible which means that it may contain entries in addition to those defined in the specification. In fact almost all objects are extensible but the fact that it is explicitly mentioned, implies that extra entries should be regarded as normal rather than exceptional. For further details see The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: Annex E, page 673.. For definitive details see:.
