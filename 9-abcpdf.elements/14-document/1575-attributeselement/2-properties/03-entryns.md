# EntryNS Property

## Notes

Represents the "NS" entry of the attribute object dictionary object.

It is defined as part of the PDF 2.0 specification.

It contains a NamespaceElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 327, page 567.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 360, page 734.
