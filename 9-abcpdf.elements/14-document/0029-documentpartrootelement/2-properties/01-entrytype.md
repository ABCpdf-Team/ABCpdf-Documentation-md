# EntryType Property

## Notes

Represents the "Type" entry of the dpartroot dictionary object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "DPartRoot".

For definitive details see:.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 408, page 829.
