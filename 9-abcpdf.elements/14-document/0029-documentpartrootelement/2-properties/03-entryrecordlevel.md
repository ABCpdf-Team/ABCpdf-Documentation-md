# EntryRecordLevel Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp int? ``` [Visual Basic] `Integer?` | null | No | Represents the "RecordLevel" entry of the dpartroot dictionary object. | 

## Notes

Represents the "RecordLevel" entry of the dpartroot dictionary object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains an integer representing a PDF numeric object.

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 408, page 829.](https://www.iso.org/standard/63534.md)
