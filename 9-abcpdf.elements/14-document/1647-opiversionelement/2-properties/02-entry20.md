## Notes

Represents the "2.0" entry of the opi version dictionary object.

It is an optional entry defined as part of the PDF 1.2 specification.

It contains a [Version20OpiElement](../../1649-version20opielement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 368, page 639.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=647)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 405, page 823.](https://www.iso.org/standard/63534.md)
