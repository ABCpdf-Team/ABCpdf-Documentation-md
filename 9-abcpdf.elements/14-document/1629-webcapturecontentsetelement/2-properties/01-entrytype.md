# EntryType Property

## Notes

Represents the "Type" entry of the web capture content set object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "SpiderContentSet".

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 352, page 621.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 388, page 801.
