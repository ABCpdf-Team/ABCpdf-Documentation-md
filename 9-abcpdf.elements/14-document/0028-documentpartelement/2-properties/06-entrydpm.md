# EntryDPM Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp Element ``` [Visual Basic] `Element` | null | No | Represents the "DPM" entry of the dpart dictionary object. | 

## Notes

Represents the "DPM" entry of the dpart dictionary object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains an [Element](../../../01-base/1086-element/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 409, page 830.](https://www.iso.org/standard/63534.md)
