# EntryParent Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp DocumentPartElement ``` [Visual Basic] `DocumentPartElement` | null | No | Represents the "Parent" entry of the dpart dictionary object. | 

## Notes

Represents the "Parent" entry of the dpart dictionary object.

It is a required entry defined as part of the PDF 2.0 specification.

It contains a [DocumentPartElement](../default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 409, page 829.](https://www.iso.org/standard/63534.md)
