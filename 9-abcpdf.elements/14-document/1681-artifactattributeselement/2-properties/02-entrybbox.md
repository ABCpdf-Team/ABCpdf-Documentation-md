# EntryBBox Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp RectangleElement ``` [Visual Basic] `RectangleElement` | null | No | Represents the "BBox" entry of the artifact attributes object. | 

## Notes

Represents the "BBox" entry of the artifact attributes object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains a [RectangleElement](../../../07-syntax/0017-rectangleelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 385, page 787.](https://www.iso.org/standard/63534.md)
