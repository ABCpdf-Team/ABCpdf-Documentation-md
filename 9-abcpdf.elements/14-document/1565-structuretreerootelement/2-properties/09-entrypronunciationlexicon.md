# EntryPronunciationLexicon Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementFileSpecificationElement> ``` [Visual Basic] `ArrayElementFileSpecificationElement>` | null | No | Represents the "PronunciationLexicon" entry of the structure tree root object. | 

## Notes

Represents the "PronunciationLexicon" entry of the structure tree root object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains an array which contains [FileSpecificationElements](../../../07-syntax/0011-filespecificationelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 322, page 557.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=565)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 354, page 720.](https://www.iso.org/standard/63534.md)
