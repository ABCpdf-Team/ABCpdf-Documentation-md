# EntryObj Property

## Notes

Represents the "Obj" entry of the object reference dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

The specification defines this property as being of any type. So any type of atom or data content is valid.

It contains an Element.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 325, page 564.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 358, page 730.
