# EntryS Property

## Notes

Represents the "S" entry of the web capture command dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a WebCaptureCommandSettingsElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 357, page 625.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 393, page 805.
