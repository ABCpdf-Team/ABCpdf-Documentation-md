# EntryCropBox Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp BoxStyleElement ``` [Visual Basic] `BoxStyleElement` | null | No | Represents the "CropBox" entry of the box colour information dictionary object. | 

## Notes

Represents the "CropBox" entry of the box colour information dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [BoxStyleElement](../../1639-boxstyleelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 360, page 630.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=638)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 396, page 810.](https://www.iso.org/standard/63534.md)
