# EntryBBox Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp RectangleElement ``` [Visual Basic] `RectangleElement` | null | No | Represents the "BBox" entry of the standard layout attributes object. | 

## Notes

Represents the "BBox" entry of the standard layout attributes object.

It is defined as part of the PDF 1.0 specification.

It contains a [RectangleElement](../../../07-syntax/0017-rectangleelement/default.md).

For definitive details see:.
