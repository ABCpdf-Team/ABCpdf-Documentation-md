# EntryTextAlign Property

## Notes

Represents the "TextAlign" entry of the standard layout attributes object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

The PDF specification states that this item assumes a value of "Start" if no value has been provided.

This item may take one of the following valid values:.

StartCenterEndJustify
