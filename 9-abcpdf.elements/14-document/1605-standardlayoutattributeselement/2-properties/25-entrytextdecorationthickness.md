| Type | Default Value | Read Only | Description |

## Notes

Represents the "TextDecorationThickness" entry of the standard layout attributes object.

It is an optional entry defined as part of the PDF 1.5 specification.

It contains a double representing a PDF numeric object.

It should have a minimum value of 0.

For definitive details see:.
