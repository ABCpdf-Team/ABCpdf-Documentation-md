# EntryMixingHints Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp DeviceNMixingHintsElement ``` [Visual Basic] `DeviceNMixingHintsElement` | null | No | Represents the "MixingHints" entry of the output intent dictionary object. | 

## Notes

Represents the "MixingHints" entry of the output intent dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a [DeviceNMixingHintsElement](../../../08-graphics/1171-devicenmixinghintselement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 365, page 633.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=641)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 401, page 816.](https://www.iso.org/standard/63534.md)
