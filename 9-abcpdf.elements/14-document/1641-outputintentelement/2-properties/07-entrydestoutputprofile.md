# EntryDestOutputProfile Property

## Notes

Represents the "DestOutputProfile" entry of the output intent dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains an ICCProfileStreamElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 365, page 634.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 401, page 815.
