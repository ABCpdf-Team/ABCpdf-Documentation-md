# EntryD Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementElement> ``` [Visual Basic] `ArrayElementElement>` | null | No | Represents the "D" entry of the box style dictionary object. | 

## Notes

Represents the "D" entry of the box style dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains [Elements](../../../01-base/1086-element/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 361, page 631.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=639)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 397, page 811.](https://www.iso.org/standard/63534.md)
