# EntryV Property

## Notes

Represents the "V" entry of the web capture information dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a double representing a PDF numeric object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 350, page 617.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 386, page 796.
