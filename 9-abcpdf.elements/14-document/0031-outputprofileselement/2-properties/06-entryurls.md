# EntryURLs Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementFileSpecificationElement> ``` [Visual Basic] `ArrayElementFileSpecificationElement>` | null | No | Represents the "URLs" entry of the destoutputprofileref dictionary object. | 

## Notes

Represents the "URLs" entry of the destoutputprofileref dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains an array which contains [FileSpecificationElements](../../../07-syntax/0011-filespecificationelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 402, page 817.](https://www.iso.org/standard/63534.md)
