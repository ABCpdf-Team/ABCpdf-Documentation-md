# EntryDefault Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp HalftoneElement ``` [Visual Basic] `HalftoneElement` | null | No | Represents the "Default" entry of the type 5 halftone dictionary object. | 

## Notes

Represents the "Default" entry of the type 5 halftone dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an [HalftoneElement](../../1316-halftoneelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 134, page 315.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=323)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 132, page 377.](https://www.iso.org/standard/63534.md)
