# EntrySC Property

## Notes

Represents the "SC" entry of the 3d cross section dictionary object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains a bool representing a PDF boolean object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 311, page 532.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 322, page 665.
