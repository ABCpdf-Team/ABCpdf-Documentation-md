# EntryF Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp FloatingWindowParametersElement ``` [Visual Basic] `FloatingWindowParametersElement` | null | No | Represents the "F" entry of the media screen parameters mh/be dictionary object. | 

## Notes

Represents the "F" entry of the media screen parameters mh/be dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a [FloatingWindowParametersElement](../../1508-floatingwindowparameterselement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 283, page 499.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=507)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 294, page 625.](https://www.iso.org/standard/63534.md)
