# EntryF Property

## Notes

Represents the "F" entry of the media screen parameters mh/be dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains a FloatingWindowParametersElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 283, page 499.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 294, page 625.
