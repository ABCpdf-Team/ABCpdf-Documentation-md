# EntryD Property

## Notes

Represents the "D" entry of the media play parameters mh/be dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a MediaDurationElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 280, page 497.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 291, page 622.
