# EntryC Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp MediaClipElement ``` [Visual Basic] `MediaClipElement` | null | No | Represents the "C" entry of the media rendition dictionary object. | 

## Notes

Represents the "C" entry of the media rendition dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [MediaClipElement](../../1500-mediaclipelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 271, page 491.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=499)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 282, page 615.](https://www.iso.org/standard/63534.md)
