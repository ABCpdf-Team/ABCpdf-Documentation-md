# EntryPL Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp MediaPlayersElement ``` [Visual Basic] `MediaPlayersElement` | null | No | Represents the "PL" entry of the media play parameters dictionary object. | 

## Notes

Represents the "PL" entry of the media play parameters dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [MediaPlayersElement](../../1510-mediaplayerselement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 279, page 496.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=504)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 290, page 621.](https://www.iso.org/standard/63534.md)
