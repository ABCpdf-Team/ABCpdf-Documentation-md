# EntryA Property

## Notes

Represents the "A" entry of the media players dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains MediaPlayerInfoElements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 290, page 502.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 301, page 629.
