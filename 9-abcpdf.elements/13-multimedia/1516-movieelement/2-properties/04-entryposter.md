This property may contain one of two different types:. 1) A bool representing a PDF boolean object. The PDF specification states that this item assumes a value of false if no value has been provided. 2) An ImageXObjectElement. For definitive details see:. The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 295, page 508. The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 306, page 635. |  |  |
