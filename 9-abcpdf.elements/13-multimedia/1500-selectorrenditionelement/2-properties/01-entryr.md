# EntryR Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementRenditionElement> ``` [Visual Basic] `ArrayElementRenditionElement>` | null | No | Represents the "R" entry of the selector rendition dictionary object. | 

## Notes

Represents the "R" entry of the selector rendition dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains [RenditionElements](../../1497-renditionelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 272, page 492.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=500)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 283, page 616.](https://www.iso.org/standard/63534.md)
