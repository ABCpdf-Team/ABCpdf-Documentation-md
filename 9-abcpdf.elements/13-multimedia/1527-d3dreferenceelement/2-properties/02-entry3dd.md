# Entry3DD Property

## Notes

Represents the "3DD" entry of the 3d reference dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a D3DStreamElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 303, page 519.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 314, page 648.
