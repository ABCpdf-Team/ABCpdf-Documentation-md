# Entry3DD Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp D3DStreamElement ``` [Visual Basic] `D3DStreamElement` | null | No | Represents the "3DD" entry of the 3d reference dictionary object. | 

## Notes

Represents the "3DD" entry of the 3d reference dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a [D3DStreamElement](../../1524-d3dstreamelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 303, page 519.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=527)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 314, page 648.](https://www.iso.org/standard/63534.md)
