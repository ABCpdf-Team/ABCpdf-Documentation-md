# EntryAN Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp D3DAnimationStyleElement ``` [Visual Basic] `D3DAnimationStyleElement` | null | No | Represents the "AN" entry of the 3d stream dictionary object. | 

## Notes

Represents the "AN" entry of the 3d stream dictionary object.

It is an optional entry defined as part of the PDF 1.7 specification.

It contains a [D3DAnimationStyleElement](../../1525-d3danimationstyleelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 300, page 516.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=524)

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.35, page 56.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=56)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 311, page 645.](https://www.iso.org/standard/63534.md)
