# EntryTT Property

## Notes

Represents the "TT" entry of the floating window parameters dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains strings, representing PDF string objects.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 284, page 501.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 295, page 627.
