# EntryT Property

## Notes

Represents the "T" entry of the media offset time dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a TimespanElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 286, page 501.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 297, page 627.
