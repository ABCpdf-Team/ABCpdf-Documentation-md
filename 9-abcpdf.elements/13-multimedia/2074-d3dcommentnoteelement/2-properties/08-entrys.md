# EntryS Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ProjectionAnnotationElement ``` [Visual Basic] `ProjectionAnnotationElement` | null | No | Represents the "S" entry of the 3d comment note object. | 

## Notes

Represents the "S" entry of the 3d comment note object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a [ProjectionAnnotationElement](../../../12-interactive/2039-projectionannotationelement/default.md).

For definitive details see:.

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.39f, page 74.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=74)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 331, page 695.](https://www.iso.org/standard/63534.md)
