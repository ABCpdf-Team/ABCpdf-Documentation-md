# EntryCT Property

## Notes

Represents the "CT" entry of the media clip data dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF string object.

This is encoded as a string using single byte ASCII encoding.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 274, page 492.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 285, page 617.
