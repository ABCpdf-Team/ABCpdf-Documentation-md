# EntryC Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp MediaCriteriaElement ``` [Visual Basic] `MediaCriteriaElement` | null | No | Represents the "C" entry of the rendition mh/be dictionary object. | 

## Notes

Represents the "C" entry of the rendition mh/be dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [MediaCriteriaElement](../../1497-mediacriteriaelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 267, page 489.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=497)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 278, page 613.](https://www.iso.org/standard/63534.md)
