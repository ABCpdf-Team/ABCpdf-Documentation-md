# EntryO Property

## Notes

Represents the "O" entry of the page object additional-actions dictionary object.

It is an optional entry defined as part of the PDF 1.2 specification.

It contains an ActionElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 195, page 416.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 198, page 505.
