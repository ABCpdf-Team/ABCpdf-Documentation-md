# EntryTrans Property

## Notes

Represents the "Trans" entry of the transition action object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a TransitionElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 215, page 429.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 219, page 521.
