# EntryType Property

## Notes

Represents the "Type" entry of the richmediainstance dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "RichMediaInstance".

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.51b, page 89.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 343, page 706.
