# EntryPosition Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp double? ``` [Visual Basic] `Double?` | null | No | Represents the "Position" entry of the collection split dictionary object. | 

## Notes

Represents the "Position" entry of the collection split dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a double representing a PDF numeric object.

For definitive details see:.

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.6b, page 31.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=31)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 158, page 450.](https://www.iso.org/standard/63534.md)
