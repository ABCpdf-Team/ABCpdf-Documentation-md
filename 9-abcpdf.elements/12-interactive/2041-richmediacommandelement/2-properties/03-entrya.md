# EntryA Property

## Notes

Represents the "A" entry of the richmediacommand dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

The specification defines this property as being of any type. So any type of atom or data content is valid.

It contains an Element.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.48b, page 41.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 223, page 524.
