# EntryI Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp DocumentInformationElement ``` [Visual Basic] `DocumentInformationElement` | null | No | Represents the "I" entry of the thread dictionary object. | 

## Notes

Represents the "I" entry of the thread dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [DocumentInformationElement](../../../14-document/1558-documentinformationelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 160, page 376.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=384)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 162, page 457.](https://www.iso.org/standard/63534.md)
