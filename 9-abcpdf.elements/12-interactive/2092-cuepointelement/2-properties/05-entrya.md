# EntryA Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ActionElement ``` [Visual Basic] `ActionElement` | null | No | Represents the "A" entry of the cuepoint dictionary object. | 

## Notes

Represents the "A" entry of the cuepoint dictionary object.

It is a required entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains an [ActionElement](../../1422-actionelement/default.md).

For definitive details see:.

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.51e, page 92.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=92)
