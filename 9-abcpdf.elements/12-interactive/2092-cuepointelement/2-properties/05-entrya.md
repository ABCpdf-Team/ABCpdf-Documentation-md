# EntryA Property

## Notes

Represents the "A" entry of the cuepoint dictionary object.

It is a required entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains an ActionElement.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.51e, page 92.
