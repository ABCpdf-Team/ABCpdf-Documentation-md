# EntryLL Property

## Notes

Represents the "LL" entry of the line annotation object.

It is defined as part of the PDF 1.6 specification.

It contains a double representing a PDF numeric object.

It should have a positive value.

The PDF specification states that this item assumes a value of 0 if no value has been provided.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 175, page 398.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 178, page 483.
