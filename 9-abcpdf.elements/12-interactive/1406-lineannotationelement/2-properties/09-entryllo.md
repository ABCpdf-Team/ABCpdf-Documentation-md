# EntryLLO Property

## Notes

Represents the "LLO" entry of the line annotation object.

It is an optional entry defined as part of the PDF 1.7 specification.

It contains a double representing a PDF numeric object.

It should have a minimum value of 0.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 175, page 399.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 178, page 483.
