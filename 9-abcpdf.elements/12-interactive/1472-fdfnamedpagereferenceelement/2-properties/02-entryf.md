# EntryF Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp FileSpecificationElement ``` [Visual Basic] `FileSpecificationElement` | null | No | Represents the "F" entry of the fdf named page reference dictionary object. | 

## Notes

Represents the "F" entry of the fdf named page reference dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [FileSpecificationElement](../../../07-syntax/0011-filespecificationelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 250, page 464.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=472)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 253, page 562.](https://www.iso.org/standard/63534.md)
