# EntryName Property

## Notes

Represents the "Name" entry of the fdf named page reference dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF string object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 250, page 464.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 253, page 562.
