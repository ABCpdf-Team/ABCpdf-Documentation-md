# EntryState Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementElement> ``` [Visual Basic] `ArrayElementElement>` | null | No | Represents the "State" entry of the set-ocg-state action object. | 

## Notes

Represents the "State" entry of the set-ocg-state action object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains [Elements](../../../01-base/1086-element/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 213, page 427.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=435)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 217, page 519.](https://www.iso.org/standard/63534.md)
