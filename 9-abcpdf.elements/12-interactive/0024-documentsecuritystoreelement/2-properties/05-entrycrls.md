# EntryCRLs Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementStreamElement> ``` [Visual Basic] `ArrayElementStreamElement>` | null | No | Represents the "CRLs" entry of the document security store dictionary object. | 

## Notes

Represents the "CRLs" entry of the document security store dictionary object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains an array which contains [StreamElements](../../../07-syntax/1028-streamelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 261, page 583.](https://www.iso.org/standard/63534.md)
