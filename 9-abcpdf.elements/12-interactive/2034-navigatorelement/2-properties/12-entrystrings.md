# EntryStrings Property

## Notes

Represents the "Strings" entry of the navigator dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a NameTreeNodeElement containing strings, representing PDF string objects.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.6d, page 34.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 160, page 454.
