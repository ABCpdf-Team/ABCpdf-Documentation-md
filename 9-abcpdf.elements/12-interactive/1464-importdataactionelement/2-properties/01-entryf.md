# EntryF Property

## Notes

Represents the "F" entry of the import-data action object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains a FileSpecificationElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 240, page 456.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 243, page 552.
