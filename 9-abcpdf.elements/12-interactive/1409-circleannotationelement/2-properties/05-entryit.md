# EntryIT Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | Represents the "IT" entry of the circle annotation object. | 

## Notes

Represents the "IT" entry of the circle annotation object.

It is an optional entry defined as part of the PDF 1.6 specification.

It contains a string representing a PDF name object.

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 177, page 401.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=409)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 180, page 486.](https://www.iso.org/standard/63534.md)
