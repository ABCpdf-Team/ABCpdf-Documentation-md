# EntryRD Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp RectangleElement ``` [Visual Basic] `RectangleElement` | null | No | Represents the "RD" entry of the circle annotation object. | 

## Notes

Represents the "RD" entry of the circle annotation object.

It is an optional entry defined as part of the PDF 1.5 specification.

It contains a [RectangleElement](../../../07-syntax/0017-rectangleelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 177, page 402.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=410)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 180, page 487.](https://www.iso.org/standard/63534.md)
