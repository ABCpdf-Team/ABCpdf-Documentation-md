# EntryBorder Property

## Notes

Represents the "Border" entry of the annotation dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains Elements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 164, page 384.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 166, page 465.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 252, page 467.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 255, page 565.
