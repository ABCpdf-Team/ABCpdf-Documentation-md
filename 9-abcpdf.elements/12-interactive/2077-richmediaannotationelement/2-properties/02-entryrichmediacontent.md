# EntryRichMediaContent Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp RichMediaContentElement ``` [Visual Basic] `RichMediaContentElement` | null | No | Represents the "RichMediaContent" entry of the richmedia annotation object. | 

## Notes

Represents the "RichMediaContent" entry of the richmedia annotation object.

It is a required entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a [RichMediaContentElement](../../2086-richmediacontentelement/default.md).

For definitive details see:.

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.49, page 77.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=77)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 333, page 696.](https://www.iso.org/standard/63534.md)
