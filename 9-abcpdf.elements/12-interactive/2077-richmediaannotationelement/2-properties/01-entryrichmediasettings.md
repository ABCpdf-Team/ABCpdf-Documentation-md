# EntryRichMediaSettings Property

## Notes

Represents the "RichMediaSettings" entry of the richmedia annotation object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a RichMediaSettingsElement.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.49, page 77.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 333, page 696.
