# EntryI Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp FormXObjectElement ``` [Visual Basic] `FormXObjectElement` | null | No | Represents the "I" entry of the appearance characteristics dictionary object. | 

## Notes

Represents the "I" entry of the appearance characteristics dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [FormXObjectElement](../../../08-graphics/1226-formxobjectelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 189, page 410.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=418)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 192, page 497.](https://www.iso.org/standard/63534.md)
