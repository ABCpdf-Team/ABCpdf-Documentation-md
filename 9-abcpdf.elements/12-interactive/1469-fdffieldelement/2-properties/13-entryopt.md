# EntryOpt Property

## Notes

Represents the "Opt" entry of the fdf field dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains Elements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 246, page 462.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 249, page 560.
