# EntryEmbeddedFDFs Property

## Notes

Represents the "EmbeddedFDFs" entry of the fdf dictionary object.

It is an optional entry defined as part of the PDF 1.4 specification.

It contains an array which contains FileSpecificationElements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 243, page 460.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 246, page 557.
