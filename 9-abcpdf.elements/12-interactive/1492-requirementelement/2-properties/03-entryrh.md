# EntryRH Property

## Notes

Represents the "RH" entry of the requirement dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains RequirementHandlerElements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 264, page 484.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 274, page 603.
