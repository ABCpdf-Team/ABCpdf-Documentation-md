# EntryMatrix Property

## Notes

Represents the "Matrix" entry of the fixed print dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a MatrixElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 191, page 411.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 194, page 498.
