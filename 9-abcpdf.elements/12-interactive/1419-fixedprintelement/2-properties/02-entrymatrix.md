# EntryMatrix Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp MatrixElement ``` [Visual Basic] `MatrixElement` | null | No | Represents the "Matrix" entry of the fixed print dictionary object. | 

## Notes

Represents the "Matrix" entry of the fixed print dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [MatrixElement](../../../07-syntax/0016-matrixelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 191, page 411.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=419)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 194, page 498.](https://www.iso.org/standard/63534.md)
