# EntryType Property

## Notes

Represents the "Type" entry of the papermetadata generation parameters dictionary object.

It is a required entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a string representing a PDF name object.

This item must always be provided and must always take the value "PaperMetaData".

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.39b, page 47.
