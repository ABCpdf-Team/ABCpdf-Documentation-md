# EntrySy Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp string ``` [Visual Basic] `string` | null | No | Represents the "Sy" entry of the caret annotation object. | 

## Notes

Represents the "Sy" entry of the caret annotation object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

The PDF specification states that this item assumes a value of "None" if no value has been provided.

This item may take one of the following valid values:.

- P
- None

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 180, page 404.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=412)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 183, page 490.](https://www.iso.org/standard/63534.md)
