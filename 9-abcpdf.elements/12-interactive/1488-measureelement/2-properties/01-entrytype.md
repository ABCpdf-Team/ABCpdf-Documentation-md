# EntryType Property

## Notes

Represents the "Type" entry of the measure dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "Measure".

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 261, page 480.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.110, page 49.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 266, page 593.
