# EntryPMD Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp PaperMetaDataGenerationParametersElement ``` [Visual Basic] `PaperMetaDataGenerationParametersElement` | null | No | Represents the "PMD" entry of the widget annotation object. | 

## Notes

Represents the "PMD" entry of the widget annotation object.

It is defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a [PaperMetaDataGenerationParametersElement](../../2047-papermetadatagenerationparameterselement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 188, page 408.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=416)

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.39, page 39.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=39)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 191, page 495.](https://www.iso.org/standard/63534.md)
