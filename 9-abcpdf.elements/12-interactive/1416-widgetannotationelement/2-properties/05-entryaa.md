# EntryAA Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp AnnotationAdditionalActionsElement ``` [Visual Basic] `AnnotationAdditionalActionsElement` | null | No | Represents the "AA" entry of the widget annotation object. | 

## Notes

Represents the "AA" entry of the widget annotation object.

It is an optional entry defined as part of the PDF 1.2 specification.

It contains an [AnnotationAdditionalActionsElement](../../1423-annotationadditionalactionselement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 188, page 409.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=417)

[Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 8.39, page 39.](http://www.adobe.com/content/dam/Adobe/en/devnet/acrobat/pdfs/adobe_supplement_iso32000.pdf#page=39)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 191, page 495.](https://www.iso.org/standard/63534.md)
