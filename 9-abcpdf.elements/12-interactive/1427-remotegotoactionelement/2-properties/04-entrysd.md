# EntrySD Property

## Notes

Represents the "SD" entry of the remote go-to action object.

It is an optional entry defined as part of the PDF 2.0 specification.

It contains a DestinationElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 200, page 419.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 203, page 509.
