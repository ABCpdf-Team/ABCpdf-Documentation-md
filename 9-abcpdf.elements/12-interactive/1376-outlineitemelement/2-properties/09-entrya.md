# EntryA Property

## Notes

Represents the "A" entry of the outline item dictionary object.

It is an optional entry defined as part of the PDF 1.1 specification.

It contains an ActionElement.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 153, page 369.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 151, page 443.
