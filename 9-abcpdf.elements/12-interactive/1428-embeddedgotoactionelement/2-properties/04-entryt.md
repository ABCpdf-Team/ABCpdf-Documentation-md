# EntryT Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp TargetElement ``` [Visual Basic] `TargetElement` | null | No | Represents the "T" entry of the embedded go-to action object. | 

## Notes

Represents the "T" entry of the embedded go-to action object.

It is defined as part of the PDF 1.0 specification.

It contains a [TargetElement](../../1428-targetelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 201, page 420.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=428)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 204, page 510.](https://www.iso.org/standard/63534.md)
