# EntryT Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementNumberFormatElement> ``` [Visual Basic] `ArrayElementNumberFormatElement>` | null | No | Represents the "T" entry of the rectilinear measure dictionary object. | 

## Notes

Represents the "T" entry of the rectilinear measure dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains [NumberFormatElements](../../1490-numberformatelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 262, page 481.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=489)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 267, page 594.](https://www.iso.org/standard/63534.md)
