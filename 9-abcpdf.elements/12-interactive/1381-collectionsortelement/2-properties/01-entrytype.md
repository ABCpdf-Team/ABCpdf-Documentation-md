# EntryType Property

## Notes

Represents the "Type" entry of the collection sort dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a string representing a PDF name object.

If provided, this item must always take the value "CollectionSort".

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 158, page 373.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 156, page 449.
