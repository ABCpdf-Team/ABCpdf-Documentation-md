# EntryActivation Property

## Notes

Represents the "Activation" entry of the richmediasettings dictionary object.

It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification.

It contains a RichMediaActivationElement.

For definitive details see:.

Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.50, page 78.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 334, page 697.
