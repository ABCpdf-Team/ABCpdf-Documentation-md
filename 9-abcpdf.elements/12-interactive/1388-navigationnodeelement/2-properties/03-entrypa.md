# EntryPA Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ActionElement ``` [Visual Basic] `ActionElement` | null | No | Represents the "PA" entry of the navigation node dictionary object. | 

## Notes

Represents the "PA" entry of the navigation node dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an [ActionElement](../../1422-actionelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 163, page 380.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=388)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 165, page 461.](https://www.iso.org/standard/63534.md)
