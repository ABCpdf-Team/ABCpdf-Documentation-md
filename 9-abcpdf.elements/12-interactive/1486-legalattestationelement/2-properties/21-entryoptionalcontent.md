# EntryOptionalContent Property

## Notes

Represents the "OptionalContent" entry of the legal attestation dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a bool representing a PDF boolean object.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 259, page 478.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 264, page 591.
