# EntryTemplates Property

## Notes

Represents the "Templates" entry of the fdf page dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains FDFTemplateElements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 248, page 463.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 251, page 561.
