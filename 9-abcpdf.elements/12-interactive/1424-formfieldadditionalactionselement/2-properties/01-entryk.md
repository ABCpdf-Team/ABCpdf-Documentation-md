# EntryK Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp JavaScriptActionElement ``` [Visual Basic] `JavaScriptActionElement` | null | No | Represents the "K" entry of the form field additional-actions dictionary object. | 

## Notes

Represents the "K" entry of the form field additional-actions dictionary object.

It is an optional entry defined as part of the PDF 1.3 specification.

It contains a [JavaScriptActionElement](../../1438-javascriptactionelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 196, page 416.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=424)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 199, page 505.](https://www.iso.org/standard/63534.md)
