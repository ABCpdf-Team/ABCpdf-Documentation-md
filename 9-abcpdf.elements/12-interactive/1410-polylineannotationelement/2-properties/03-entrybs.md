# EntryBS Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp BorderStyleElement ``` [Visual Basic] `BorderStyleElement` | null | No | Represents the "BS" entry of the polyline annotation object. | 

## Notes

Represents the "BS" entry of the polyline annotation object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains a [BorderStyleElement](../../1394-borderstyleelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 178, page 402.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=410)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 181, page 488.](https://www.iso.org/standard/63534.md)
