# EntryBE Property

## Notes

Represents the "BE" entry of the polyline annotation object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an Element.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 178, page 403.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 181, page 488.
