| Represents the "HAlign" entry of the richmediaposition dictionary object. It is an optional entry defined as part of the PDF 1.7 Extension Level 3 specification. It contains a string representing a PDF name object. The PDF specification states that this item assumes a value of "Far" if no value has been provided. This item may take one of the following valid values:. NearCenterFar For definitive details see:. Adobe Supplement to the ISO 32000, BaseVersion: 1.7, ExtensionLevel: 3; Table: 9.50f, page 85.
