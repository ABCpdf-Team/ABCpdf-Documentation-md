# EntryColorants Property

## Notes

Represents the "Colorants" entry of the devicen colour space attributes dictionary object.

It is defined as part of the PDF 1.0 specification.

It contains an Element.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 71, page 162.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 70, page 204.
