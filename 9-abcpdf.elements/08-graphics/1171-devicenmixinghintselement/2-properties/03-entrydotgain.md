# EntryDotGain Property

## Notes

Represents the "DotGain" entry of the devicen mixing hints dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an Element.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 73, page 164.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 72, page 206.
