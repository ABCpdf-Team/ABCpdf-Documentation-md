# EntryOCGs Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementOptionalContentGroupElement> ``` [Visual Basic] `ArrayElementOptionalContentGroupElement>` | null | No | Represents the "OCGs" entry of the usage application dictionary object. | 

## Notes

Represents the "OCGs" entry of the usage application dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an array which contains [OptionalContentGroupElements](../../1230-optionalcontentgroupelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 103, page 233.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=241)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 101, page 286.](https://www.iso.org/standard/63534.md)
