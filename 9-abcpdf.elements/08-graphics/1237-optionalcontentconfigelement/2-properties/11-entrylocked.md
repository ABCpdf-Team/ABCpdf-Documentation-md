# EntryLocked Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp ArrayElementOptionalContentGroupElement> ``` [Visual Basic] `ArrayElementOptionalContentGroupElement>` | null | No | Represents the "Locked" entry of the optional content configuration dictionary object. | 

## Notes

Represents the "Locked" entry of the optional content configuration dictionary object.

It is an optional entry defined as part of the PDF 1.6 specification.

It contains an array which contains [OptionalContentGroupElements](../../1230-optionalcontentgroupelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 101, page 231.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=239)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 99, page 283.](https://www.iso.org/standard/63534.md)
