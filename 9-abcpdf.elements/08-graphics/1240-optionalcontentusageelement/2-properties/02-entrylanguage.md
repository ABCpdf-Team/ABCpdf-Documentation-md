# EntryLanguage Property

| Type | Default Value | Read Only | Description | 
| --- | --- | --- | --- |
| **[C#]** ```csharp OptionalContentUsageLanguageElement ``` [Visual Basic] `OptionalContentUsageLanguageElement` | null | No | Represents the "Language" entry of the optional content usage dictionary object. | 

## Notes

Represents the "Language" entry of the optional content usage dictionary object.

It is an optional entry defined as part of the PDF 1.0 specification.

It contains an [OptionalContentUsageLanguageElement](../../1240-optionalcontentusagelanguageelement/default.md).

For definitive details see:.

[The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 102, page 232.](https://opensource.adobe.com/dc-acrobat-sdk-docs/standards/pdfstandards/pdf/PDF32000_2008.pdf#page=240)

[The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 100, page 285.](https://www.iso.org/standard/63534.md)
