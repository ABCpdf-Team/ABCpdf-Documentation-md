# EntryOCGs Property

## Notes

Represents the "OCGs" entry of the optional content properties dictionary object.

It is a required entry defined as part of the PDF 1.0 specification.

It contains an array which contains OptionalContentGroupElements.

For definitive details see:.

The ISO PDF Specification, ISO 32000-1:2008 PDF 1.7; Table: 100, page 228.

The ISO PDF Specification, ISO 32000-2:2017 PDF 2.0; Table: 98, page 281.
